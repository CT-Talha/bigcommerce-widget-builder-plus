Your Store Hash

The store hash is a short code that identifies your BigCommerce store. It's in your admin URL.

📍 Where to find it
https://store-XXXXX.mybigcommerce.com/manage

Log into your BigCommerce admin. Look at the URL in your browser — the part after store- and before .mybigcommerce.com is your store hash.

Looks like a short alphanumeric string — no spaces

API Access Token

An API token lets this tool securely connect to your store. Create one in your BigCommerce admin.

🔑 How to create an API token
  1. Go to your BigCommerce admin → Settings → API Accounts
  2. Click Create API Account → Create V2/V3 API Token
  3. Give it a name (e.g. Widget Tool)
  4. Under OAuth Scopes, find Content and set it to Modify
  5. Click Save — copy the token shown on the next screen
⚠ The token is only shown once — copy it before closing the page
